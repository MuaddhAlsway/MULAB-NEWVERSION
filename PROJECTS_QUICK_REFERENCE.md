## 🛠️ How to Modify Projects

### To Edit a Project's Data:
1. Open `/src/app/data/projects.ts`
2. Find the project by ID (1-44)
3. Modify these fields as needed:
   - `name` - Project title
   - `description` - Short description
   - `longDescription` - Full details
   - `tech` - Technology array
   - `category` - fullstack, htmlcssjs, htmlcss, or c
   - `image` - Image path in /public/Projects/
   - `github`, `live`, `linkedin` - Links
   - `features`, `challenges`, `learnings` - Details

### To Update Project Image:
1. Place new image in `/public/Projects/` folder
2. Update the `image` field in projects.ts:
   ```typescript
   image: '/Projects/NewImageName.png'
   ```

### To Add a New Project:
1. Add to projects.ts with ID (45 or higher)
2. Add corresponding image to `/public/Projects/`
3. Fill in all required fields (id, name, description, tech, category, image, github, live, linkedin)
4. Image will automatically display and project will be filterable

### To Remove a Project:
1. Delete the project object from projects.ts array
2. (Optional) Delete the image from /public/Projects/
